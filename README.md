# 工作计划助手 — 任务计划管理 PWA

移动端优先的工作任务管理应用，专为基层工作场景设计。

## 功能

- 📝 **极速记录** — 一键记录领导口头交代的任务，3秒完成
- 🔔 **补全提醒** — 未补全详情的任务自动提醒
- ⚠️ **智能预警** — 长周期任务防拖延，超期自动标红
- ☀️ **早晚打卡** — 定时推送今日待办和完成回顾
- 📤 **数据备份** — JSON 导出/导入，数据完全在本地

## 使用方式

### 方法一：本地 HTTP 服务器（最简单）

```bash
cd task-planner
python -m http.server 8080
```

手机连接同一 WiFi，浏览器访问 `http://<电脑IP地址>:8080`

### 方法二：部署到服务器

将 `task-planner/` 文件夹内容上传到任意静态文件服务器（Nginx、IIS、GitHub Pages 等）。

## 添加到手机主屏幕

1. 用手机浏览器打开应用网址
2. Chrome/Edge: 菜单 → "添加到主屏幕"
3. Safari: 分享按钮 → "添加到主屏幕"

添加后打开即为全屏应用体验，可离线使用。

## 数据说明

所有任务数据存储在浏览器本地（IndexedDB），不经过任何第三方服务器。使用 📤 导出功能定期备份数据。

## 技术栈

- 纯 HTML/CSS/JavaScript，无框架
- IndexedDB 本地存储
- Service Worker 离线缓存 + 推送通知
- PWA Manifest 可安装

## 文件说明

| 文件 | 用途 |
|------|------|
| index.html | 主应用（单文件，内嵌 CSS + JS） |
| sw.js | Service Worker（离线缓存 + 通知） |
| manifest.json | PWA 清单 |
| icon.svg | 应用图标 |
